📏 Runbook: CLS Ratio (Cumulative Layout Shift)

Alerte: CLS élevé (Core Web Vitals)
Seuil: > 0.1 (mobile/desktop)
Impact: Expérience utilisateur dégradée

📋 Description

Le CLS mesure la stabilité visuelle de la page. Un CLS élevé (>0.1) signifie que les éléments de la page se déplacent après le chargement, ce qui frustre les utilisateurs.

🔍 Causes fréquentes

  • Images sans dimensions - Pas de width/height définis
  • Polices web - Flash de texte invisible (FOIT)
  • Publicités/iframes - Injections dynamiques
  • Contenu injecté en JS - Ajout après rendu
  • Animations CSS - Qui modifient le layout

🛠️ Solutions

Images

<!-- Toujours spécifier width/height -->
<img src="image.jpg" width="800" height="600" alt="...">

<!-- Ou en CSS -->
img {
    width: 100%;
    height: auto;
    aspect-ratio: 16/9;
}

Polices

/* Optimiser le chargement des polices */
@font-face {
    font-family: 'CustomFont';
    src: url('/fonts/custom.woff2') format('woff2');
    font-display: swap;  /* Évite FOIT */
}

Réserver l'espace pour contenu dynamique

/* Réserver un espace pour les pubs/iframes */
.ad-container {
    min-height: 250px;
    width: 100%;
}
