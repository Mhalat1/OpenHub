ℹ️ Runbook: PHP Info

Alerte: Information sur la configuration PHP
Type: Monitoring

📋 Configuration PHP actuelle

Pour vérifier la config:

# Créer temporairement un fichier info.php
<?php
phpinfo();
?>

# En ligne de commande
php -i
php -m  # Modules chargés

# Fichier de config
php --ini

Paramètres importants à vérifier:

Directive Valeur recommandée Vérification
memory_limit 256M php -i | grep memory_limit
max_execution_time 30 php -i | grep max_execution_time
upload_max_filesize 20M php -i | grep upload_max
post_max_size 20M php -i | grep post_max
opcache.enable 1 php -i | grep opcache

📦 Extensions requises

# Vérifier les extensions installées
php -m

# Extensions communes nécessaires:
- PDO
- PDO_MySQL / PDO_PgSQL
- mbstring
- json
- curl
- openssl
- fileinfo
- gd (images)
- xml
- zip
- redis / memcached
